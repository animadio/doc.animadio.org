1 - Variables

1.1 - Why CSS variables ?

The only reason Animadio is fully written in CSS is the presence of custom properties & CSS variables.

Custom properties have the particularity of being overloaded directly from the custom stylesheet file...

It is therefore possible to completely customize any element of the framework thanks to the CSS variables, in order to adapt it to each use case !

1.2 - How to use CSS variables ?

Just choose the required variable with its selector in the following file :

Variables

Then in your custom stylesheet file, write a CSS declaration with the necessary selector, the custom property to modify & the value you want !

You can find more informations about custom properties (CSS variables) usage here :

Custom Properties Var Function Variable Usage

2 - Keyframes

2.1 - Where Keyframes would be used ?

Keyframes could be used by 3 ways :

  1. in some Elements classes
  2. in the Anima classes
  3. in your own stylesheet file

2.2 - How to use Keyframes ?

Just choose the required keyframes with its name in the following file :

Keyframes

Then in your custom stylesheet file, write a CSS declaration with the necessary selector, the animation property, the keyframes name to use & the animation options you want !

You can find more informations about keyframes usage here :

Keyframes Animation Animation Name Keyframes Usage

3 - Init

3.1 - What Init does ?

Init is here to manage 3 kind of things :

  1. hide the 2 main classes : check & radio
  2. define on main tags important properties with certain values without the possibility to change them
  3. set custom properties to all necessary tags

3.2 - How to change there values ?

Just choose the required variable with its selector in the following file :

Init

Then in your custom stylesheet file, write a CSS declaration with the necessary selector, the custom property to modify & the value you want !

You can find more informations about the definitions of HTML tags here :

HTML Elements

4 - Box

4.1 - What's inside the Box ?

Box file contains 5 types of helper classes handling the Box Model :

  1. the Container classes
  2. the Margin classes
  3. the Padding classes
  4. the Height classes
  5. the Width classes

4.2 - Usage of the Box Classes

For the Container classes, you need to concat the class name "container-" with the size you want : between 70, 80 & 90, in a specific media : from xs to xl.

For the other box classes, you need to concat the full class name like "mar-all-none", or "pad-top-md", or "height-sm", "max-w-lg" with the keyword "base" or "global" : the first to get the effect on the element, the second to get the effect on the direct children.

5 - Helpers

5.1 - What kind of helpers ?

The 3 types of helper classes available in addition are :

  1. the Font classes
  2. the Text classes
  3. the Cursor classes
